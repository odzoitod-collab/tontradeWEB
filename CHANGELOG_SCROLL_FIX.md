# 📝 Changelog: Исправление скролла

## Версия 2.0.2 - 26 января 2026

### 🐛 Исправлена проблема со скроллом

**Проблема:**
После добавления блокировки pull-to-refresh, страница перестала нормально листаться вверх.

**Решение:**
Реализована умная блокировка, которая:
- ✅ Блокирует pull-to-refresh только в самом верху страницы
- ✅ Разрешает нормальный скролл вверх и вниз
- ✅ Разрешает скролл внутри контейнеров
- ✅ Проверяет, находится ли элемент в скроллируемом контейнере

### Технические детали

#### Старая логика (❌ неправильно)
```javascript
if (preventPullToRefresh && touchYDelta > 0) {
  e.preventDefault(); // Блокирует ВСЕ движения вниз
}
```

#### Новая логика (✅ правильно)
```javascript
// Проверяем:
// 1. Тянем вниз?
// 2. Страница в самом верху?
// 3. Не внутри скроллируемого контейнера?
if (touchDelta > 0 && window.scrollY === 0 && !isScrollable) {
  // Дополнительная проверка на скроллируемый элемент
  if (!canScroll) {
    e.preventDefault(); // Блокируем только pull-to-refresh
  }
}
```
