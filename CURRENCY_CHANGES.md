# Изменения валюты по умолчанию

## Что изменено:

### 1. Основная валюта по умолчанию
- **Было**: RUB (российский рубль)
- **Стало**: USD (доллар США)

### 2. Файлы изменены:

#### `utils/currency.ts`
```typescript
// Было:
export const DEFAULT_CURRENCY = 'RUB';

// Стало:
export const DEFAULT_CURRENCY = 'USD';
```

#### `App.tsx`
```typescript
// Обновлен комментарий при регистрации:
preferred_currency: DEFAULT_CURRENCY, // USD по умолчанию
```

#### `components/WalletPage.tsx`
```typescript
// Конвертер валют теперь начинается с USD:
const [convertFrom, setConvertFrom] = useState('USD');
```

## Результат:

✅ При первом заходе на сайт пользователи увидят цены в долларах  
✅ Новые пользователи регистрируются с USD как основной валютой  
✅ Конвертер валют по умолчанию показывает USD  
✅ Все расчеты прибыли/убытка отображаются в долларах  

## Проверка:
Сервер запущен на http://localhost:3001/ - можно проверить изменения в браузере.
